Game of Life

Ce projet pédagogique basé sur Conway's Game of Life est écrit en Rust et compilé en WebAssembly avec les outils wasm-bindgen et wasm-pack.

Utiliser l'option de build --target web me permet de directement importer le fichier wasm dans un module javascript sans utiliser de bundler.
